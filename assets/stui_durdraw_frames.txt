============================================================
  DURDRAW IMPLEMENTATION NOTES
============================================================

CANVAS SETUP:
  - Canvas size: 10 cols Г— 6 rows (gives breathing room)
  - Braille block: top-left anchor at col 3, row 1
  - Wordmark "STUI": row 5, col 3, letter-spacing 1
  - Background: color index for black
  - Foreground: color index for your nearest Tyrian purple
    (xterm-256: color 99 = #875fff or 105 = #8787ff)
    If using truecolor Durdraw: #9B5DE5

LAYER STRUCTURE:
  Layer 1 (bottom) вҖ” background fill
  Layer 2 вҖ” braille play button block
  Layer 3 вҖ” wordmark "STUI" (starts blank, types in on frame 15)

LOOPING:
  - Set animation to loop from frame 01 back after frame 18
  - Total runtime: ~3.5s per loop

BUFFERING VARIANT:
  - On frame 10 (full hold), overlay "buffering..." text
    below the braille block instead of the wordmark
  - Switch to wordmark only on frame 14 (ready state)

XTERM-256 PURPLE CANDIDATES:
  57  = #5f00ff  (deep violet)
  93  = #8700ff  (purple)
  99  = #875fff  (soft purple)  вҶҗ closest to Tyrian
  135 = #af5fff  (light purple)
  141 = #af87ff  (lavender)
